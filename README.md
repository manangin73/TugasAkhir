# Sistem Peminjaman Alat Musik 🎵

## 📌 Overview
Aplikasi berbasis web untuk mengelola peminjaman alat musik secara efisien, termasuk booking, manajemen user, dan tracking status peminjaman.

## 🚀 Features
- User authentication (login/register)
- Booking alat musik
- Manajemen data alat
- Riwayat peminjaman
- Admin dashboard

## 🛠 Tech Stack
- Laravel
- MySQL
- Bootstrap / Tailwind (sesuaikan)
